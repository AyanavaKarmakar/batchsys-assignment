## Live Deploy Link: https://batchsys-assignment.vercel.app/

### Tech Used:

1. Next.js
2. TypeScript
3. Tailwind CSS
4. Zustand
5. Radix Icons
6. Heroicons

## Getting Started

1. Clone this repository

```shell
git clone https://github.com/AyanavaKarmakar/batchsys-assignment.git
```

2. Install dependencies

```shell
npm install
```

3. Run the app

```shell
npm run dev
```
